# Atari 2600 Ms. Pac-Man — DQN 강화학습

> 인게임 이미지 기반 DQN(Deep Q-Network) 강화학습 연구 프로젝트

---

## 프로젝트 개요

Atari 2600의 Ms. Pac-Man 환경에서 **인게임 이미지(픽셀 데이터)를 입력**으로 사용하는 DQN 강화학습을 구현하고, 학습 과정의 시각화 및 행동 유도 방식에 대한 연구를 진행하였습니다.

---

## 주요 연구 내용

### 1. 인게임 이미지 기반 학습
- 게임 화면의 원시 픽셀 데이터를 수정하여 입력으로 활용
- 별도의 게임 상태 정보 없이 이미지만으로 학습 수행

### 2. 이미지 재구축 및 Influence Map 시각화
- 게임 이미지를 재구축하여 학습 과정을 직관적으로 시각화
- **Influence Map**을 활용해 에이전트의 의사결정 패턴 분석 및 학습이 의도된 방향으로 진행되도록 유도

### 3. 학습 로그 분석 (`graph_generator`)
- 학습 진행 상황 로그를 추출하여 그래프로 시각화
- 에피소드별 보상, 손실 등 주요 지표를 분석

---

## 한계 및 아쉬운 점

| 항목 | 내용 |
|------|------|
| 보너스 아이템 처리 | 게임 내 과일 등 보너스 점수 아이템에 대한 대응이 미흡 |
| 키 입력 지연 | 연속 키 입력으로 인해 추론 결과와 실시간 게임 상황이 불일치하는 경우 발생 |
| 코너 회전 | 벽에 부딪히는 경우를 제외하고 코너를 원활하게 돌지 못하는 문제 존재 |

---

## 기술 스택

- **환경**: Atari 2600 (OpenAI Gym / ALE), Python==3.8, Pycharm
- **알고리즘**: DQN (Deep Q-Network)
- **시각화**: Influence Map, graph_generator
